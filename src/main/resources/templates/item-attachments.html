Item Attachment

Resource for Attachments of an Item, typically images for an Item.

Operations

  • POST rest/v1/trade-memberships/{tradeMembershipId}/items/{itemId}/attachments/{attachmentId}
  • GET rest/v1/trade-memberships/{tradeMembershipId}/items/{itemId}/attachments/
  • DELETE rest/v1/trade-memberships/{tradeMembershipId}/items/{itemId}/attachments/

Authorization

  • All operations can be performed by clients with a valid Authorization header
