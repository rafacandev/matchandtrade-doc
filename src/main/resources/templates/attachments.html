Attachments

Resource for Attachments uploaded to the system. Example, item pictures are uploaded via this resource and then associated to the item via item attachments.

Operations

  • POST rest/v1/attachments

Authorization

  • All operations can be performed by clients with a valid Authorization header

Resource

Property Name Format Description
contentType read-only, 150 max length The Content Type of this Attachment, this value extracted when the Attachment is uploaded to the system
attachmentId read-only Id associated with this resource
_links read-only

Links for versions of the Attachment (mostly essence files).

  • original: the uploaded essence file on its original form
  • thumbnail: thumbnail generated from the original essence file
name read-only The name of the Attachment, this value is extracted when the Attachment is uploaded to the system
