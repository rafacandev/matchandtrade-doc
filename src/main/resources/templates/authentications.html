Authentications

Resource for Authentications. It returns information about the current authenticated session, typically used to get the userId associated with the current session.

Operations

  • GET rest/v1/authentications

Authorization

  • Only clients with a valid Authorization header

Resource

Property Name Format Description
userId read-only The User associated with this authentication, this value is created during the sign-up (first successful Authenticate)

Example

${AUTHENTICATIONS_PLACEHOLDER}
