Offer

Resource for Offer. An Offer is a proposal to exchange an offered Item for a wanted Item.

Operations

  • POST /rest/v1/trade-memberships/{tradeMembershipId}/offers/
  • GET /rest/v1/trade-memberships/{tradeMembershipId}/offers/{offerId}
  • GET /rest/v1/trade-memberships/{tradeMembershipId}/offers?{queryParameters}
  • DELETE /rest/v1/trade-memberships/{tradeMembershipId}/offers/{offerId}

Authorization

  • All operations can be performed by clients with a valid Authorization header

Resource

Property Name Format Description
offerId read-write, mandatory Id associated with this resource
offeredItemId ready-only Id associated with the offered Item
wantedItemId ready-only Id associated with the wanted Item in exchange for the Offer.offeredItemId

Query Parameters

${PAGINATION_TABLE}

Rules

  • Offer.offeredItemId and Offer.wantedItemId must differ.
  • Offer.offeredItemId and Offer.wantedItemId must belong existing Items.
  • Offer.offeredItemId must belong to the offering User.userId when performing a POST or DELETE operations.
  • Offer.offeredItemId and Offer.wantedItemId must be associated to the same Trade.
