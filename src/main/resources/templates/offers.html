Offer

Resource for Offer. An Offer is a proposal to exchange an offered Article for a wanted Article.

Operations

  • POST /rest/v1/memberships/{membershipId}/offers/
  • GET /rest/v1/memberships/{membershipId}/offers/{offerId}
  • GET /rest/v1/memberships/{membershipId}/offers?{queryParameters}
  • DELETE /rest/v1/memberships/{membershipId}/offers/{offerId}

Authorization

  • All operations can be performed by clients with a valid Authorization header

Resource

Property Name Format Description
offerId read-write, mandatory Id associated with this resource
offeredArticleId ready-only Id associated with the offered Article
wantedArticleId ready-only Id associated with the wanted Article in exchange for the Offer.offeredArticleId

Query Parameters

${PAGINATION_ROWS}
Property Name Description
offeredArticleId Refers to a Offer.offeredArticleId
wantedArticleId Refers to a Offer.wantedArticleId

Rules

  • Offer.offeredArticleId and Offer.wantedArticleId must differ.
  • Offer.offeredArticleId and Offer.wantedArticleId must belong existing Articles.
  • Offer.offeredArticleId must belong to the offering User.userId when performing a POST or DELETE operations.
  • Offer.offeredArticleId and Offer.wantedArticleId must be associated to the same Trade.
