Search

Endpoint to perform advanced searches. Its usage differs according to the given search.recipe property.

Operations

  • POST rest/v1/search

Authorization

  • "search.recipe": "ITEMS"Only authenticated clients

Resource Common to all Searches

Property Name Format Description
recipe mandatory, enumeration (see Recipes) The search recipe to be used. Each recipe may return different response bodies and support different request bodies
criteria See Criteria Resource The search criteria for this search

Criteria

Property Name Format Description
key mandatory, string Values differ according to the search.recipe
value mandatory, number or string according to the current search.recipe Possible values and types differ according to the search.recipe

Recipes

Recipe Name Description Criteria Specification
ITEMS Returns an array of Items matching the given search.criteria "key": "trade.tradeId" matching trade.tradeId

"value": n numeric value matching trade.tradeId
