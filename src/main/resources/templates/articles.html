Article

Resource for Articles. Article is anything that a User would put for up for Trade.

Operations

  • POST rest/v1/memberships/{membershipId}/articles/
  • PUT rest/v1/memberships/{membershipId}/articles/
  • GET rest/v1/memberships/{membershipId}/articles/{articleId}
  • GET rest/v1/memberships/{membershipId}/articles?{queryParameters}
  • GET rest/v1/memberships/{membershipId}/articles/
  • DELETE rest/v1/memberships/{membershipId}/articles/{articleId}

Authorization

  • All operations can be performed by clients with a valid Authorization header
  • The URL parameter {membershipId} must be associated to the authenticated client's userId

Resource

Property Name Format Description
articleId read-only Id associated with this resource
name read-write, mandatory, unique, 3 min length, 150 max length Name of this Article
description read-write, optional, 500 max length Description of this Article

Query Parameters

${PAGINATION_TABLE}

Rules

  • Articles can only be deleted by their owners.
  • Article.name must be unique (case insensitive) within a Membership.
