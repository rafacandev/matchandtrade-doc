Item

Resource for Items. Item is anything that a User would put for up for Trade.

Operations

  • POST rest/v1/trade-memberships/{tradeMembershipId}/items/
  • PUT rest/v1/trade-memberships/{tradeMembershipId}/items/
  • GET rest/v1/trade-memberships/{tradeMembershipId}/items/{itemId}
  • GET rest/v1/trade-memberships/{tradeMembershipId}/items?{queryParameters}
  • GET rest/v1/trade-memberships/{tradeMembershipId}/items/
  • DELETE rest/v1/trade-memberships/{tradeMembershipId}/items/{itemId}

Authorization

  • All operations can be performed by clients with a valid Authorization header
  • The URL parameter {tradeMembershipId} must be associated to the authenticated client's userId

Resource

Property Name Format Description
itemId read-only Id associated with this resource
name read-write, mandatory, unique, 3 min length, 150 max length Name of this Item
description read-write, optional, 500 max length Description of this Item

Query Parameters

Property Name Description
_pageNumber See pagination
_pageSize See pagination

Rules

  • Items can only be deleted by their owners.
  • Item.name must be unique (case insensitive) within a TradeMembership.
