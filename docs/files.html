Files

Resource for Files. Files uploaded to the system. Example, item pictures are uploaded via this resource and then associated to the item via Item Files.

Operations

  • POST rest/v1/files

Authorization

  • All operations can be performed by clients with a valid Authorization header

Resource

Property Name Format Description
contentType read-only, 150 max length The Content Type of this File, this value extracted when the file is uploaded to the system
fileId read-only Id associated with this resource
_links read-only

Links for versions of the File (aka file essence).

  • original: the uploaded file in its original form
  • thumbnail: thumbnail generated from the original file
name read-only The name of the File, this value extracted when the file is uploaded to the system

Query Parameters

Property Name Description
_pageNumber See pagination
_pageSize See pagination
