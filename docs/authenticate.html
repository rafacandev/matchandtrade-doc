Authenticate

Endpoint to authenticate on the system. It redirects the client to an oAuth server (e.g.: Google oAuth) and returns with a header called Authorization.

The Authorization header acts as a session token and needs to be included to any secured URL.

Operations

  • GET authenticate
  • GET authenticate/sign-out

Authorization

  • Public

Examples

----- Request ----- GET http://localhost:8081/authenticate ----- Response ----- Status: HTTP/1.1 200 Headers: {Authorization: 6278-3904-6690}
----- Request ----- GET http://localhost:8081/authenticate/sign-out Headers: {Authorization: 6278-3904-6690} ----- Response ----- Status: HTTP/1.1 205
