Authenticate

_Endpoint_ for authentication. It redirects the client to an oAuth server and returns back to the client with a header called Authorization.

The `Authorization` header acts as a session token and needs to be included to any secured URL.

Operations

  • GET authenticate
  • GET authenticate/sign-out

Authorization

  • Public

Examples

----- Request ----- GET http://localhost:8081/authenticate ----- Response ----- Status: HTTP/1.1 200 Headers: {Authorization: 9022-7062-8927}
----- Request ----- GET http://localhost:8081/authenticate/sign-out Headers: {Authorization: 9022-7062-8927} ----- Response ----- Status: HTTP/1.1 205
