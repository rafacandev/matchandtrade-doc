Authenticate

Endpoint to authenticate on the system. It redirects the client to an oAuth server (e.g.: Google oAuth) and returns with a header called Authorization. Additionally, it creates a Cookie: JSESSIONID allowing requests to authenticate/info which also returns the Authorization header.

The Authorization header acts as a session token and needs to be included to any URL witch requires authorization.

Operations

  • GET authenticate
  • GET authenticate/info
  • GET authenticate/sign-out

Authorization

  • Public endpoints: authenticate and authenticate/sing-out
  • Only requests containing Cookie: JSESSIONID are authorized to access authenticate/info
