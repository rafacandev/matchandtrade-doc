ItemFile

Resource for Files of an Item, typically images for an Item.

Operations

  • POST rest/v1/trade-memberships/{tradeMembershipId}/items/{itemId}/files/{fileId}
  • GET rest/v1/trade-memberships/{tradeMembershipId}/items/{itemId}/files/

Authorization

  • All operations can be performed by clients with a valid Authorization header

Resource

See File documentation.

Query Parameters

Property Name Description
_pageNumber See pagination
_pageSize See pagination
