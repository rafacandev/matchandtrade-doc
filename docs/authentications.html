Authentications

_Resource_ for authentications. It returns information about the current authenticated session, typically used to get the userId associated with the current session.

Operations

Authorization

  • Only authenticated clients

Resource

Property Name Format Description
userId read-only The user associated with this authentication, this value is created during the sign-up (first successful authenticate)

Example

----- Request ----- GET http://localhost:8081/rest/v1/authentications/ Headers: {Authorization: 9396-644-7794}{Content-Type: application/json} ----- Response ----- Status: HTTP/1.1 200 { "userId" : 4, "_links" : [ { "rel" : "self", "href" : "http://localhost:8081/rest/v1/authentications/" } ] }
