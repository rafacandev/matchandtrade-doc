TradeResult

Resource for TradeResults. TradeResults is a report which describes how Items should be traded according to the Offers in place.

Operations

  • GET rest/v1/trades/{tradeId}/results

Authorization

  • All operations can be performed by clients with a valid Authorization header
  • The URL parameter {tradeId} must be associated to the authenticated client's userId

Resource

Property Name Format Description
itemId read-only Id associated with this resource
name read-write, mandatory, unique, 3 min length, 150 max length Name of this Item

Query Parameters

Property Name Description
_pageNumber See pagination
_pageSize See pagination

Examples

For example, the row 1,Olavo,488,Apples to Apples,:RECEIVES:,2,Maria,492,Caylus,:SENDS:,3,Xavier translates to: Olavo receives Maria's Caylus and Olavo sends his Apples to Apples to Xavier
