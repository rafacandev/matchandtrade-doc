WantItem

Resource for WantItem. A WantItem is an Item that wants be exchanged for another Item. Example: the URL GET /rest/v1/trade-memberships/1/items/2/want-items means: get all the Items that want to be exchanged for the Item.itemId: 2.

Operations

  • POST /rest/v1/trade-memberships/{tradeMembershipId}/items/{itemId}/want-items/
  • GET /rest/v1/trade-memberships/{tradeMembershipId}/items/{itemId}/want-items/{wantItemId}
  • GET /rest/v1/trade-memberships/{tradeMembershipId}/items/{itemId}/want-items?{queryParameters}
  • DELETE /rest/v1/trade-memberships/{tradeMembershipId}/items/{itemId}/want-items/{wantItemId}

Authorization

  • Only authenticated clients

Resource

Property Name Format Description
itemId read-write, mandatory Must belong to an existing Item within the same Trade but within a different TradeMembership
priority read-write, mandatory, integer, 1 min value, 1000 max value Priority for this WantItem. The lower the priority the more wanted the Item is
wantItemId ready-only Id associated with this resource

Query Parameters

Property Name Description
_pageSize See pagination
_pageNumber See pagination

Rules

  • WantItem.itemId must belong to an existing Item within the Trade associated with the trade-memberships/{tradeMembershipId}. This concept my be confusing at first, see Use Cases for a demonstration.
